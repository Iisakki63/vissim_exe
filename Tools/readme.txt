COMMAND LINE TOOLS

Using the avr_delays.py tool:
python avr_delays.py out/XXX.rsr out/YYY.del

Using the veh_counts.py tool:
python veh_count.py out/XXX.fhz links.txt out/YYY.cnt
The links.txt file maps the link numbers with more descriptive labels

Note: the test_input files have additional x in the file name
so that version management would not remove them.
